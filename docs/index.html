📚 TaskCIn - Gerenciador de Tarefas Acadêmicas

TaskCIn é um sistema de gerenciamento de tarefas acadêmicas que utiliza a matriz de Eisenhower para classificar atividades com base em sua importância e urgência.

🎯 Visão Geral

O sistema permite o gerenciamento de diversos tipos de atividades acadêmicas:

  • Tarefas: genéricas, com descrição livre
  • Provas: com matéria associada
  • Projetos: com matéria e nível de complexidade
  • Relatórios: com matéria e plataforma de entrega

Todas as atividades são automaticamente organizadas na Matriz de Eisenhower.

🧪 Tecnologias Utilizadas

  • Backend: C++ (lógica de negócios) + Node.js (API REST)
  • Frontend: HTML, CSS e JavaScript
  • Persistência: Arquivos JSON

🖥️ Estrutura do Projeto

  • frontend/: Interface do usuário
  • backend/: Servidor, lógica e dados
  • backend/include/: Headers das classes C++
  • backend/source/: Implementações das classes
  • backend/data/: Armazenamento em todos.json

⚙️ Pré-requisitos

  • Node.js
  • npm
  • Compilador C++
  • Git

🚀 Como Executar

  1. Clone o repositório
  2. Compile o código C++:
    cd backend/source
    g++ todo_storage.cpp Tarefa.cpp Prova.cpp Projeto.cpp Relatorio.cpp -o ../todo_storage
  3. Inicie o servidor com:
    npm start
  4. Acesse http://localhost:3000 no navegador

🧩 Funcionalidades

  • 📊 Matriz de Eisenhower para priorização
  • 🎓 Suporte a diferentes tipos de atividades acadêmicas
  • 📝 Interface intuitiva para adicionar, editar e excluir tarefas
  • 🔄 Atualização em tempo real
